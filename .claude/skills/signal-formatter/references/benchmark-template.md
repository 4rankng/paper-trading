# Benchmark Comparison Template

Use this template for all signals:

```markdown
## Benchmark Comparison: {TICKER} vs SPY/QQQ

**SPY (S&P 500):**
- 6M Return: X%
- Trend: [UPTREND/DOWNTREND/SIDEWAYS]
- Technical: [Strong/Weak/Bullish/Bearish]
- Volatility (ATR %): Y%

**QQQ (Nasdaq-100):**
- 6M Return: A%
- Trend: [UPTREND/DOWNTREND/SIDEWAYS]
- Technical: [Strong/Weak/Bullish/Bearish]
- Volatility (ATR %): B%

**{TICKER} vs Benchmarks:**
- Expected Alpha: Z% over SPY/QQQ
- Driver Edge: Popularity (fleeting) or Substance (durable)?
- Risk-Adjusted Edge: [Superior/Inferior/Neutral]
- Thesis Justification: [Specific catalyst/edge]
- **Recommendation:** [BUY STOCK / BUY SPY / BUY QQQ / HOLD]
```

## How to Complete

1. Get SPY/QQQ 6M returns from analytics or market data
2. Analyze trend (price vs MA50/MA200)
3. Compare volatility (ATR % or standard deviation)
4. Calculate expected alpha for your ticker
5. Classify edge as "Popularity" (sentiment) or "Substance" (fundamentals)
6. Assess risk-adjusted superiority
7. Make final recommendation
